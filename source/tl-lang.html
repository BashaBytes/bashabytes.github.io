TL_LANG

This is the official page for TL_LANG

TL_LANG is a toy programming language written in Python

Installation

git clone https://github.com/BashaBytes/TL_LANG.git
cd TL_LANG
python3 main.py # For running the example.tll file, just edit the example.tll file as required
python3 gui.py # For GUI version of the interpreter

You can also try out your code on Online Interpreter of TL_LANG

Example Code:

anuko a = 5
chupinchu a

Output:

5

Documentation

'#' is used for commenting in the language.

Variables

Variables can be declared using anuko.

anuko a = 30
anuko b = 50
chupinchu a + b

Output:

80

Simple Looping

Loops can be created using looplo keyword.

looplo 2
chupinchu "namaste"

Output:

namaste
namaste
